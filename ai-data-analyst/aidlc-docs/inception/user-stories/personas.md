# User Personas — AI 데이터 분석가 챗봇 (프로토타입)

## Persona 1: 데모 운영자 (Demo Operator)
- **역할**: AWS Summit 워크샵에서 챗봇을 시연하는 팀원
- **목표**: 사전 준비된 질문 예시(20~30개)를 입력하여 파이프라인 동작을 검증
- **기술 수준**: 개발자 (시스템 내부 구조 이해)
- **사용 패턴**: 미리 정의된 질문을 순서대로 입력, 결과 확인
- **핵심 니즈**: 모든 질문에 대해 차트 이미지 + 설명 + 테이블이 정상 출력되는지 확인

## Persona 2: 데모 관객 (Demo Audience)
- **역할**: AWS Summit에서 데모를 관람하는 참석자
- **목표**: 자연어 질문이 시각화 결과로 변환되는 과정을 이해
- **기술 수준**: 비개발자 (데이터 분석 비전문가)
- **사용 패턴**: 데모 운영자가 입력하는 질문과 결과를 관람
- **핵심 니즈**: 직관적인 차트와 한국어 설명으로 데이터 인사이트를 빠르게 파악
